# Boxes


Deployed at https://lucas-test.github.io/boxes/

## Help

In the input zone:

    x y z
    w1 h1 d1 w2 h2 d2  ...

(x y z) is the coordinate of the corner of the stair.
(w1 h1 d1) (w2 h2 d2) are the dimension of the boxes.
